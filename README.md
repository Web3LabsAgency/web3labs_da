# 🧪 Web3 Labs | Agencia Digital EdTech

Bienvenido al repositorio oficial de **Web3 Labs**. Somos una agencia digital enfocada en la educación y adopción de tecnologías descentralizadas. Nuestra plataforma sirve como el puente principal para que nuevos usuarios y emprendedores se integren al ecosistema Web3 de manera fluida, segura y con bases sólidas.

## 🎯 Visión del Proyecto
Nuestra misión es democratizar el acceso al conocimiento sobre blockchain y criptomonedas. Este website no solo funciona como el portal de nuestros servicios educativos, sino que sirve como una demostración técnica de la tecnología Web3, utilizando la blockchain de Hive como nuestra infraestructura de backend y pasarelas de pago cripto nativas.

## 📚 Servicios que Ofrecemos
* **🚀 Onboarding Emprendedor a tecnología Web3:** Acompañamiento guiado para negocios y creadores que desean transicionar o integrar herramientas descentralizadas.
* **₿ Curso Integral sobre Bitcoin:** Formación profunda sobre los fundamentos, la filosofía, la seguridad y el impacto económico de Bitcoin.
* **🤝 Consultorías Personalizadas:** Sesiones 1-a-1 adaptadas a las necesidades específicas de cada usuario, proyecto o modelo de negocio.

## ✨ Características Principales de la Plataforma
* **Ramp-On Automático (Fiat a Cripto):** Los usuarios pueden pagar nuestros servicios utilizando sus monedas locales (fiat) mediante tarjeta de crédito/débito. Web3 Labs recibe los fondos de forma automática en criptomonedas.
* **Pagos Nativos con Exchanges:** Integración directa con las APIs de **Binance Pay** y **Bitget Pay** para usuarios que ya operan dentro de estos ecosistemas.
* **Interfaz de Usuario Adaptable:** Soporte nativo para modo **Claro / Oscuro** (Dark/Light mode) para mejorar la experiencia de lectura y navegación.
* **Backend Descentralizado:** Todo el registro de operaciones, acceso a cursos y gestión de contenido interactúa con **Hive Blockchain** (hive.io), garantizando transparencia, velocidad y cero comisiones de red por transacción.

## 🛠️ Stack Tecnológico
Todo el núcleo del proyecto está construido bajo el ecosistema de **Python**.
* **Lenguaje Principal:** Python 3.10+
* **Framework Web:** [FastAPI / Django / Flask] (Para el manejo de rutas y APIs).
* **Interacción Blockchain:** Librería `beem` para Python, conectando el website con los nodos RPC de Hive.
* **Frontend:** Plantillas renderizadas desde Python (Jinja2) / HTML5, CSS3, con toggle de tema claro/oscuro mediante JavaScript puro.
* **Pasarelas de Pago:** APIs oficiales de Binance, Bitget y proveedores de Ramp-On (ej. Transak, MoonPay).
